# 로봇팔–라즈베리파이–키오스크 통신 구현 프로젝트

이 저장소에는 안드로이드 기반 키오스크에서 동작하도록 제작된 APK 파일이 포함되어 있습니다.  
해당 애플리케이션은 사용자가 키오스크 화면에서 주문을 입력하면,  
이를 라즈베리파이(Raspberry Pi)로 전달하여 로봇팔(Robot Arm)을 제어하는 통신 구조를 구현하는 것을 목표로 개발되었습니다.

완성에는 이르지 못했지만, 통신 구조 설계와 프로토타입 개발을 통해  
시스템 간 연동 과정에서 발생하는 여러 기술적 문제를 직접 경험하며 많은 것을 배운 프로젝트입니다.


## APK 소개

- 안드로이드형 키오스크에 설치되는 전용 앱입니다.  
- 안드로이드 키오스크에서 버튼 입력 시 USB–TTL 기반 시리얼 메시지를 라즈베리파이에 전송하도록 설계했습니다.  
- 라즈베리파이는 전달받은 문자열 명령을 파싱하여 로봇팔 제어 코드로 넘기는 구조를 목표로 하였습니다.


## 시스템 구조(시도한 설계)

Android Kiosk -(USB-TTL Serial)-> Raspberry Pi -> Robot Arm

### 구성 요소
- 키오스크 앱 (APK) : 주문 이벤트 발생 → 시리얼 메시지 전송  
- 라즈베리파이 : 문자열 기반 명령 파싱 → 모터 제어 코드 실행  
- 로봇팔 : MG996R 서보 기반 또는 ROS2 로봇팔 제어 방식 적용
