# 열매 수확 로봇을 구현한 캡스톤 프로젝트 파일들입니다

# Capstone_Arduino_Code  
아두이노 코드들로 이루어져 있으며, 여러 가지 센서들을 제어하기 위한 코드가 포함되어 있습니다.  
라인 센서, 거리 센서, 모터 제어 등 로봇의 하드웨어 동작을 담당하는 부분을 구현하였습니다.

# Capstone_Python_Code  
파이썬 코드들로 구성되어 있으며, 아두이노와의 시리얼 통신 및 YOLOv5 학습 코드가 들어 있습니다.  
라즈베리파이에서 카메라 입력을 받아 객체 감지 모델을 실행하고, 인식 결과를 아두이노로 전달하는 역할을 수행합니다.

# Matlab_Kinematic  
매트랩을 사용하여 로봇팔의 기구학(Kinematics)을 계산한 데이터가 포함되어 있습니다.  
링크 길이 기반 위치 계산, 역기구학 시도 등 로봇팔 동작을 분석하기 위한 자료가 포함되어 있습니다.
